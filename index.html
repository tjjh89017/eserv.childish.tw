Welcome to eserv Project

eserv: Lightweight embedded server-side native C/C++ Web Framework
It needs your help to improve it.

eserv: 輕量級嵌入式原生C/C++網頁伺服器框架
他需要您來加強他的功能 

$ cd eserv/lib/eserv && make clean all
$ cd ../../ && make clean all
$ ./eserv

What does it need?

We need to rebuild the code about the Libevent parts, and make sure the code could be extended for the future.

我們必須要重構Libevent部份的程式碼，且要確定能夠容易維護並且擴充。
而近期目標是可以跟ARM作業系統結合，成為嵌入式系統簡單的作業系統以及網頁系統控制框架，提供路由器等等的網路設備基礎以及相關API
